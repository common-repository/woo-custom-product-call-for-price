Plugin extends WooCommerce by outputting "Call for Price" when product enable for "Call for Price"

== Description ==

Call for Price for WooCommerce plugin extends WooCommerce by outputting "Call for Price" when product enable for "Call for Price".

== Plugin Features ==

- here we add custom product
- add custom tab for it and you may select product for the " Call For Price"
- you may enter you phone number and other information
- this information display on frontside of "Call for price" product list page
- user click on "Call for pay" and get your mobile number with other information which fill up you in admin-side


== Installation ==

1. Upload the entire Custom Product Type For WooCommerce - Call For Price' folder to the '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. manually you select your product for set "call for price "
